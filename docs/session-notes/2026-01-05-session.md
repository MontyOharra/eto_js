## Cleanup Requirements Established

From `docs/CODEBASE_CLEANUP_CHECKLIST.md`:

1. **Python 3.10+ typing**: Use `T | None` instead of `Optional[T]`, `list[T]` instead of `List[T]`
2. **Extract Literal types**: Define named type aliases like `FilterRuleField = Literal["a", "b"]`
3. **DTOs as pure data**: No behavior flags like `clear_errors` - use function parameters instead
4. **No defaults in entity dataclasses**: Callers must provide values explicitly
5. **Defaults OK in Create DTOs**: For truly optional fields with sensible defaults
6. **Module functions over static methods**: For stateless utilities
